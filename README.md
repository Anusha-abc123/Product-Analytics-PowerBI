# Product Analytics Dashboard (Power BI)

## Overview
This project showcases a multi-page Power BI dashboard built to analyze product performance for a food-delivery platform.

## Dashboard Pages

### 1. Product Overview
- Key KPIs: Total Revenue, Total Users, Total Orders, AOV
- Monthly revenue and order trends
- City-wise order distribution

### 2. Funnel & Conversion
- User conversion funnel from signup to first order
- Drop-off analysis to identify friction points

### 3. Retention & Channels
- Active users trend over time
- Users by acquisition channel
- Revenue contribution by channel

## Key Insights
- Some acquisition channels generate higher revenue despite lower user volume
- Significant post-signup drop-offs indicate onboarding optimization opportunities
- Retention trends highlight scope for re-engagement campaigns

## Tools Used
- Power BI
- DAX
- Data Modeling & Visualization

## Use Case
Designed to support product, growth, and business teams in data-driven decision-making.
